React.js | HTML5 | Javascript | TypeScript | ChakraUI

# Installation and Setup
1. Clone the repository:
- For frontend
```bash
git clone `https://github.com/Hemant142/MovieWatchList/` 
```
```bash
cd movie-watch-list
```
install the dependencies
```
npm install
```
run project
```
npm run start
```

run db.json server
```
npm run server
```
